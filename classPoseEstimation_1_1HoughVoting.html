Estimates the rigid transformation of the source to the target point cloud using the correspondences of descriptors calculated at the keypoints of both point clouds.

Parameters

targetThe target point cloud.
source_keypointsThe keypoints of the source point cloud at which descriptors were calculated.
target_keypointsThe keypoints of the target point cloud at which descriptors were calculated.
source_featuresThe descriptors of the source point cloud.
target_featuresThe descriptors of the target point cloud.

transformationsThe resulting 4x4 transformation matrix of the estimated rigid transformation.
Returns
Whether the estimation was successfull.
